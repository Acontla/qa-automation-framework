## Descripción

Este proyecto es un framework de automatización de pruebas diseñado para demostrar habilidades en **Quality Assurance (QA)**. Utiliza el patrón **Page Object Model (POM)** para mantener el código limpio y fácil de mantener.

###  Objetivo
- Demostrar conocimientos en automatización de pruebas
- Implementar buenas prácticas de testing
- Crear un portafolio profesional de QA

---

## Tecnologías utilizadas

| Tecnología | Versión | Propósito |
|------------|---------|-----------|
| Python | 3.9+ | Lenguaje de programación |
| Selenium WebDriver | 4.15 | Automatización de navegador |
| Pytest | 7.4 | Framework de pruebas |
| WebDriver Manager | 4.0 | Gestión automática de drivers |
